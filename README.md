# Rover Controller App
A laptop app for controlling robotic rovers via WiFi

The following commands are available as push buttons as well as keyboard hotkeys:

"Forward" - W <br>
"Backward" - S <br>
"TurnLeft" - Q <br>
"TurnRight" - E <br>
"RotateLeft" - A <br>
"RotateRight" - D <br>
"Release" - N/A <br>
"Claw-Open" - SPACE <br>
"Claw-Close" - SPACE <br>
