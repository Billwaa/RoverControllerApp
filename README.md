# Rover Controller App
A laptop app for controlling robotic rovers via WiFi

The following commands are available as push buttons as well as keyboard hotkey:

"Forward" - W
"Backward" - S
"TurnLeft" - Q
"TurnRight" - E
"RotateLeft" - A
"RotateRight" - D
"Release" - N/A
"Claw-Open" - SPACE
"Claw-Close" - SPACE
